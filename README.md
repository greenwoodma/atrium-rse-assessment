# ATRIUM RSE Assessment Task

This repository consists of a [single Jupyter Notebook](./notebook.ipynb) which contains my solution to the ATRIUM RSE Assessment Task.

Note that I've completed both Option A (Item Type Conversion) and Option B (Duplicate Detection) and both solutions are contained within the single notebook along with general discussion of the SSH Open Marketplace API.

Also included is [a modified copy of the SSH Open Marketplace OpenAPI description](./api-docs.json). I've needed to produce this version as [the official published version](https://marketplace-api.sshopencloud.eu/v3/api-docs) unfortunatley contains a logical flaw which means it can't be used to generate a validating OpenAPI client. See the notebook for more details on the flaw and my suggested modifications.
